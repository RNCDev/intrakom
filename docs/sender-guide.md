# Using the Sender

The sender is a web page — no installation needed. Open it in any browser on any device connected to your home network.

---

## Open the sender

In any browser, go to:
```
http://<hub-ip>:8000
```

Replace `<hub-ip>` with the IP address of the device running the hub (e.g. `http://192.168.1.10:8000`).

> **Tip:** Bookmark this URL for easy access. On a phone, you can add it to your home screen: tap the share button → "Add to Home Screen".

---

## Allow microphone access

The first time you open the sender, your browser will ask for microphone permission. Click **Allow**.

- **Chrome/Edge**: A prompt appears at the top of the page
- **Safari (iOS)**: Tap **Allow** in the system dialog
- **Firefox**: A prompt appears in the address bar

If you accidentally denied permission, see [Troubleshooting](troubleshooting.md).

---

## Talk

1. You'll see buttons for each receiver (room) that is currently online
2. **Press and hold** a room button to talk — the button turns red while you're transmitting
3. **Release** the button to stop
4. The person in that room will hear you through their speakers

### Talking to all rooms at once

If two or more receivers are online, an **All** button appears. Hold it to talk to every room simultaneously.

### Busy rooms

If someone else is already talking to a room, you'll see a "busy" notice. Wait for them to release the button and try again.

---

## Status indicators

- **Green dot + "Connected"** — sender is connected to the hub, ready to talk
- **Grey dot + "Offline"** — sender has lost connection to the hub (check your network)
- Room button is dim/greyed out — that receiver is offline (the device isn't running the receiver app)

---

## On a phone or tablet

The sender works on phones and tablets. For the best experience:
- Use Chrome on Android
- Use Safari on iPhone/iPad
- Keep the browser tab open and screen awake while in use

> **iOS Safari note:** iOS requires a user gesture (tap) before the microphone can be used. Tap the page first, then hold a room button.
